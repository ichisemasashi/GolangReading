switch caseは、上から下へcaseを評価します。 
caseの条件が一致すれば、そこで停止(自動的にbreak)します。

(例えば、

\begin{lstlisting}[numbers=none]
switch i {
case 0:
case f():
}
\end{lstlisting}

では、 \texttt{i==0} であれば、 \texttt{case 0} でbreakされるため
\texttt{f} は呼び出されません)
